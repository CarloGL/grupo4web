Presentación del Grupo 4

Este es nuestro proyecto de implementación de un sistema de login utilizando tecnologías modernas como:

  • Node.js para el backend
  • JWT para la autenticación segura
  • Bootstrap para el diseño responsive
  • HTML5 y CSS3 para la estructura y estilos

Desarrollado como parte de nuestra presentación académica, demostrando la implementación de un sistema de autenticación moderno y seguro.
