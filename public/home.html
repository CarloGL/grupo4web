Implementación de MongoDB Atlas

Base de Datos: MongoDB Atlas (Plan Free)

¿Por qué elegimos MongoDB Atlas?

  • Base de datos NoSQL en la nube con un generoso plan gratuito
  • Fácil integración con Node.js a través de Mongoose
  • Interfaz visual intuitiva para gestionar datos
  • Escalabilidad y alta disponibilidad sin costo adicional

Proceso de Implementación

Nos registramos en MongoDB Atlas y creamos una cuenta gratuita que nos ofrece:
  • 512 MB de almacenamiento
  • Compartido RAM
  • Backend serverless
  • Multi-región disponible

  1. Creamos un nuevo cluster en la región más cercana
  2. Configuramos el acceso IP para permitir conexiones
  3. Creamos usuario y contraseña para la base de datos
  4. Obtuvimos la URI de conexión para nuestro proyecto

Implementamos la conexión utilizando:
  • Mongoose como ODM (Object Data Modeling)
  • Variables de entorno para credenciales seguras
  • Modelos para estructura de datos de usuarios
  • Métodos de autenticación y validación

Aprendizajes Clave

Durante la implementación de MongoDB Atlas, aprendimos:

  • Gestión de bases de datos NoSQL en la nube
  • Configuración de seguridad y acceso a bases de datos
  • Modelado de datos con Mongoose
  • Buenas prácticas en el manejo de credenciales
